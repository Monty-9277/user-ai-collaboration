# 协同场景完整检查清单

- [ ] 已完整扫描现有代码/文档（避免空地起大楼）
- [ ] 已确认所有需求维度（功能/性能/限制/偏好）

- [ ] 已本地/最小样本测试验证

## 场景D：破坏性操作（删除/环境修改）

- [ ] 已列出"即将删除/修改"清单
